# Privacy Policy – VIT Relative Grade Estimator

VIT Relative Grade Estimator is a Chrome extension designed to help VIT students
estimate relative grades using marks displayed on the VTOP website.

## Data Collected
- Academic marks and course-related information visible on the VTOP Marks View page

## Data Usage
- Used solely to calculate and display relative grade estimates
- No authentication credentials, passwords, or financial information are collected
- Data is not used for advertising, tracking, or profiling

## Data Sharing
- Data is not sold, shared, or transferred to third parties
- Data is processed only for the extension’s core functionality
